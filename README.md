# PocketCli

PocketCli is a lightweight DevOps toolkit.

Features

- SSH launcher
- automatic server discovery
- lightweight monitoring
- automatic provisioning

Works on

- Linux
- macOS
- WSL
- Alpine
- iSH (iPad)

Install

curl -fsSL https://raw.githubusercontent.com/YOURORG/pocketcli/main/install.sh | sh

Run

pocketcli
